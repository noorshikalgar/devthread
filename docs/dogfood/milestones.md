# Dogfood Milestones

## M0: Task-Thread Loop

Entry: accepted thread-first direction and working macOS toolchain.

Exit:

- Create/select a task and set status/next step.
- Add typed updates with `Cmd+Enter` in under five seconds.
- Retain failed drafts and reopen the selected task after restart.
- Edit entries, inspect immutable revisions, restore a revision, and undo trash.
- Pass TypeScript, frontend unit, Rust repository, and production build checks.

## M1: Two-Week Personal Dogfood

Entry: M0 exit criteria met.

Exit:

- Record capture friction, resume usefulness, missing context, and unused UI.
- Decide whether Inbox, Today/resume, search, or automatic backup is the next
  highest-value capability.
- Confirm the product is used as a work journal rather than a todo list.

## Later Milestones

Only sequence these after M1 evidence: Inbox/Today, search/reports, automatic
backup/export, smart context, integrations, web/sync, public website, and
cross-platform distribution.
